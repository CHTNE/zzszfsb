# 高级技巧

## Frontmatter 高级用法

### 排序控制

使用 `order` 字段精确控制文档在同分类中的排列顺序：

```yaml
---
order: 1    # 数字越小越靠前
---
```

### 置顶文章

使用 `pin: true` 将重要文章置顶，在首页和分类中都会优先展示：

```yaml
---
pin: true
---
```

### 标签系统

为文档添加标签，方便分类和搜索：

```yaml
---
tags: [前端, React, TypeScript]
---
```

## 搜索技巧

- 使用 `Ctrl+K` (Mac: `Cmd+K`) 快速打开搜索
- 支持模糊搜索标题、标签和内容
- 使用方向键导航搜索结果
- 按 `Enter` 打开文档，按 `Esc` 关闭

## 主题切换

Wiki 支持三种主题模式：

1. **亮色模式** - 适合白天使用
2. **暗色模式** - 护眼，适合夜间使用
3. **跟随系统** - 自动匹配系统设置

点击右上角的主题按钮即可切换。

## 自定义样式

通过修改 `app/globals.css` 中的 CSS 变量来自定义主题：

```css
:root {
  --background: #ffffff;
  --foreground: #171717;
  --link-color: #0969da;
}
```
