# 基础用法

## 创建文档

在 `docs/` 目录下直接创建 `.md` 文件即可。文件名会自动作为文档标题。

## 文件夹分类

使用文件夹来组织文档分类：

```
docs/
├── 快速开始/
│   ├── 介绍.md
│   └── 安装.md
├── 指南/
│   └── 基础用法.md
└── API/
    └── 接口文档.md
```

## Markdown 语法支持

### 代码块

支持多种语言的代码高亮：

```javascript
function hello() {
  console.log("Hello, Wiki!");
}
```
