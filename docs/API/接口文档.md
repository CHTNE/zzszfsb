# 接口文档

## 文档配置接口

每个 MD 文件支持以下 Frontmatter 配置：

```typescript
interface DocFrontmatter {
  title?: string;       // 自定义标题，默认使用文件名
  order?: number;       // 排序权重，默认 999
  pin?: boolean;        // 是否置顶，默认 false
  tags?: string[];      // 标签列表
  description?: string; // 描述信息
}
```

## 目录结构

```typescript
interface DocCategory {
  name: string;         // 分类名称
  path: string;         // 分类路径
  files: DocFile[];     // 分类下的文档
  children: DocCategory[]; // 子分类
  totalCount: number;   // 总文档数
}
```

## 文档数据

```typescript
interface DocFile {
  title: string;        // 文档标题
  slug: string;         // URL 路径
  tags: string[];       // 标签
  description: string;  // 描述
  content: string;      // Markdown 内容
  headings: Heading[];  // 标题列表
}
```

## API 路由

| 路由 | 说明 |
|------|------|
| `/` | Wiki 首页 |
| `/docs/[...slug]` | 文档详情页 |
