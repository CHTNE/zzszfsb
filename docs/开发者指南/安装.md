## 环境要求

- Node.js 18+
- npm 或 yarn

## 安装步骤

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

启动后访问 `http://localhost:3000` 即可看到 Wiki 首页。

### 3. 构建生产版本

```bash
npm run build
npm start
```

## 项目结构

| 目录/文件 | 说明 |
|-----------|------|
| `docs/` | Markdown 文档目录 |
| `app/` | Next.js 页面 |
| `components/` | React 组件 |
| `lib/` | 工具函数 |

## 常见问题

### Q: 如何添加新文档？

在 `docs/` 目录下创建 `.md` 文件即可，支持多级文件夹分类。

### Q: 支持哪些 Markdown 语法？

支持 GFM (GitHub Flavored Markdown)，包括表格、任务列表、代码高亮等。
