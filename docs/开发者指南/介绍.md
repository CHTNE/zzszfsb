这是一个基于 **Next.js** 构建的现代化 Wiki 知识库站点。你只需要在 `docs/` 目录下编写 Markdown 文件，即可自动添加到 Wiki 中。

![知识库 Wiki 概念图](/images/wiki-demo.png)

## 主要功能

- **自动扫描** - 新增 MD 文件后，开发服务器会自动识别并展示
- **文件夹分类** - 通过文件夹结构自动组织文档分类
- **Frontmatter 元数据** - 支持标题、排序、标签、置顶等配置
- **全文搜索** - 按 `Ctrl+K` 快速搜索文档内容
- **暗色主题** - 支持亮色/暗色主题切换
- **响应式设计** - 完美适配桌面和移动端

## 快速开始

1. 在 `docs/` 目录下创建文件夹作为分类
2. 在文件夹内创建 `.md` 文件
3. 可选：在文件头部添加 frontmatter 元数据
4. 启动开发服务器即可预览

## Frontmatter 示例

```yaml
---
title: 自定义标题
order: 1          # 排序权重
pin: true         # 置顶
tags: [前端, Vue]  # 标签
description: 简要描述
---
```

> 提示：没有 frontmatter 的文件也能正常使用，系统会自动使用文件名作为标题。

## 插入图片

支持两种图片插入方式：

**Markdown 语法：**
```md
![图片描述](/images/your-image.png)
```

**HTML 标签（支持自定义宽度）：**
```html
<img src="/images/your-image.png" alt="描述" width="600" />
```

将图片文件放到 `public/images/` 目录下即可引用。
